# 1a-preCaTT: Raw Pre-Syntax for preCaTT

This module defines the raw (untyped) pre-syntax of preCaTT. It is the
computational backbone for the later intrinsically typed layers.

The guiding principle here is:
- constructors are only the syntax (`Ctx`, `Var`, `Ty`, `Tm`, `Sub`)
- operations such as weakening, substitution, composition, and identity are
  *defined functions* that compute by structural recursion

This gives a small trusted core and makes later proofs easier, because most
equations reduce by computation instead of requiring extra rewriting axioms.

```agda
module 1a-preCaTT where

open import Relation.Binary.PropositionalEquality
  using (_≡_ ; refl ; trans ; sym ; cong ; cong₂ ; subst)
open import Data.Bool.Base using (Bool; true; false; _∨_; _∧_)
open import 0a-Logic
```

## Data Declarations

This is the raw syntax with de Bruijn variables.

Types are generated by:
1. `⋆` for objects (0-cells)
2. hom-types `[ A ] t ⇒ u`

Terms are either variables or raw coherences with an explicit stored
substitution. Substitutions are plain snoc-lists (`∅S`, `⟨_,_⟩`) at this raw
layer.

```agda
infixl 5 _,_
infixl 6 [_]_⇒_

data Ctx : Set
data Var : Ctx → Set
data Ty  : Ctx → Set
data Sub : Ctx → Ctx → Set
data Tm  : Ctx → Set

data Ctx where
  -- Empty context.
  ∅   : Ctx
  -- Context extension by a type that may mention earlier variables.
  _,_ : (Γ : Ctx) → Ty Γ → Ctx

data Var where
  -- Most recent variable.
  vz : ∀ {Γ A} → Var (Γ , A)
  -- Weaken an older variable across a new binder.
  vs : ∀ {Γ A} → Var Γ → Var (Γ , A)

data Ty where
  ⋆       : ∀ {Γ} → Ty Γ
  [_]_⇒_  : ∀ {Γ} (A : Ty Γ) (t u : Tm Γ) → Ty Γ

data Sub where
  ∅S    : ∀ {Γ} → Sub Γ ∅
  ⟨_,_⟩ : ∀ {Γ Δ A} (σ : Sub Γ Δ) (t : Tm Γ) → Sub Γ (Δ , A)

data Tm where
  -- Variable term.
  var     : ∀ {Γ} (x : Var Γ) → Tm Γ
  -- Raw coherence with explicit base type, endpoints, and substitution.
  coh-raw : ∀ {Γ Δ} (A : Ty Δ) (u v : Tm Δ) (σ : Sub Γ Δ) → Tm Γ
```

## Weakening Operations and Identity Substitution

Weakening is defined by mutual structural recursion over the syntax tree.
The design choice is that weakening is an operation, not a constructor: later
proofs can therefore compute with it directly instead of appealing to extra
equational axioms.

```agda
mutual
  -- Weakening on types acts recursively on the base type and endpoints.
  wkTy : ∀ {Γ A} → Ty Γ → Ty (Γ , A)
  wkTy ⋆ = ⋆
  wkTy ([ B ] t ⇒ u) = [ wkTy B ] wkTm t ⇒ wkTm u

  -- Weakening on terms shifts variables and weakens stored substitutions.
  wkTm : ∀ {Γ A} → Tm Γ → Tm (Γ , A)
  wkTm (var x) = var (vs x)
  wkTm (coh-raw B u v σ) = coh-raw B u v (wkSub σ)

  -- Weakening a substitution weakens each stored term.
  wkSub : ∀ {Γ Δ A} → Sub Γ Δ → Sub (Γ , A) Δ
  wkSub ∅S = ∅S
  wkSub ⟨ σ , t ⟩ = ⟨ wkSub σ , wkTm t ⟩
  
-- Identity substitution extends recursively with the newest variable term.
idS : (Γ : Ctx) → Sub Γ Γ
idS ∅       = ∅S
idS (Γ , A) = ⟨ wkSub (idS Γ) , var vz ⟩
```

## Variable Typing

Extract the type of a variable from the context.
Because variables are de Bruijn indices, this computation is completely
structural: `vz` reads off the newest declaration, and `vs x` delegates to the
smaller variable and weakens the result.

```agda
var-to-type : ∀ {Γ} → Var Γ → Ty Γ
-- The newest variable has the last declared type, weakened into the full context.
var-to-type {Γ = _ , A} vz     = wkTy A
-- Older variables keep their original type, weakened across the new binder.
var-to-type {Γ = _ , A} (vs x) = wkTy (var-to-type x)
```

## Substitution, Composition, and Lookup

Substitution and composition are mutually recursive, computing by structural
recursion on the syntax. This is the core computational content of the module:
all later algebraic laws are proved about these definitions.

```agda
infixl 8  _[_]T _[_]t
infixr 10 _∘_

mutual
  -- Lookup interprets a variable through a substitution.
  lookup : ∀ {Γ Δ} → Var Δ → Sub Γ Δ → Tm Γ
  -- The newest variable maps to the last term in the snoc-list.
  lookup vz     ⟨ σ , t ⟩ = t
  -- Older variables are looked up in the tail.
  lookup (vs x) ⟨ σ , t ⟩ = lookup x σ

  -- Type substitution recursively substitutes in the base type and endpoints.
  _[_]T : ∀ {Γ Δ} → Ty Δ → Sub Γ Δ → Ty Γ
  ⋆ [ σ ]T = ⋆
  ([ B ] t ⇒ u) [ σ ]T = [ B [ σ ]T ] t [ σ ]t ⇒ u [ σ ]t

  -- Term substitution computes by lookup for variables.
  _[_]t : ∀ {Γ Δ} → Tm Δ → Sub Γ Δ → Tm Γ
  var x [ σ ]t = lookup x σ
  -- Coherences keep their shape and compose the stored substitution.
  coh-raw B u v τ [ σ ]t = coh-raw B u v (τ ∘ σ)

  -- Substitution composition is snoc-wise.
  _∘_ : ∀ {Γ Δ Θ} → Sub Δ Θ → Sub Γ Δ → Sub Γ Θ
  ∅S ∘ σ = ∅S
  ⟨ τ , t ⟩ ∘ σ = ⟨ τ ∘ σ , t [ σ ]t ⟩
```

## Type-of Function

```agda
tyOf : ∀ {Γ} → Tm Γ → Ty Γ
-- Variables are typed by lookup in the context.
tyOf (var x)             = var-to-type x
-- A coherence has the substituted hom-type of its stored data.
tyOf (coh-raw B u v σ)  = ([ B ] u ⇒ v) [ σ ]T
```

## Structural Congruence Lemmas

These lemmas establish that the raw syntax satisfies the expected equational theory of a CwF.
The proofs are grouped by role: first lookup facts, then functoriality of
substitution, then identity laws, and finally the weakening/substitution
interaction needed later.

### Step 2.1: Lookup Helper Lemmas

```agda
abstract
  lookup-wkSub : ∀ {Γ Δ A} (x : Var Δ) (σ : Sub Γ Δ)
               → lookup x (wkSub {A = A} σ) ≡ wkTm (lookup x σ)
  lookup-wkSub vz     ⟨ σ , t ⟩ = refl
  lookup-wkSub (vs x) ⟨ σ , t ⟩ = lookup-wkSub x σ
```

```agda
abstract
  -- lookup commutes with composition
  lookup-∘ : ∀ {Γ Δ Θ} (x : Var Θ) (τ : Sub Δ Θ) (σ : Sub Γ Δ)
           → lookup x (τ ∘ σ) ≡ lookup x τ [ σ ]t
  lookup-∘ vz     ⟨ τ , t ⟩ σ = refl
  lookup-∘ (vs x) ⟨ τ , t ⟩ σ = lookup-∘ x τ σ
```

```agda
abstract
  -- lookup of identity substitution gives the variable term
  lookup-idS : ∀ {Γ} (x : Var Γ) → lookup x (idS Γ) ≡ var x
  lookup-idS {Γ = Γ , A} vz     = refl
  lookup-idS {Γ = Γ , A} (vs x) =
    trans (lookup-wkSub x (idS Γ)) (cong wkTm (lookup-idS x))
```

### Step 2.2: Functoriality (composition distributes over substitution)

```agda
abstract
  mutual
    [∘]T : ∀ {Γ Δ Θ} (A : Ty Θ) (τ : Sub Δ Θ) (σ : Sub Γ Δ)
         → A [ τ ]T [ σ ]T ≡ A [ τ ∘ σ ]T
    [∘]T ⋆ τ σ = refl
    [∘]T ([ B ] t ⇒ u) τ σ =
      cong₂ (λ X Y → [ B [ τ ]T [ σ ]T ] X ⇒ Y) ([∘]t t τ σ) ([∘]t u τ σ)
        |> λ p → trans (cong₂ (λ X Y → [ X ] t [ τ ]t [ σ ]t ⇒ Y)
                               ([∘]T B τ σ) ([∘]t u τ σ))
                       (cong (λ X → [ B [ τ ∘ σ ]T ] X ⇒ u [ τ ∘ σ ]t)
                             ([∘]t t τ σ))
      where _|>_ : ∀ {a b : Set} → a → (a → b) → b
            x |> f = f x

    [∘]t : ∀ {Γ Δ Θ} (t : Tm Θ) (τ : Sub Δ Θ) (σ : Sub Γ Δ)
         → t [ τ ]t [ σ ]t ≡ t [ τ ∘ σ ]t
    [∘]t (var x)             τ σ = sym (lookup-∘ x τ σ)
    [∘]t (coh-raw B u v γ)  τ σ = cong (coh-raw B u v) (assocS γ τ σ)

    assocS : ∀ {Γ Δ Θ Ξ} (γ : Sub Θ Ξ) (τ : Sub Δ Θ) (σ : Sub Γ Δ)
           → (γ ∘ τ) ∘ σ ≡ γ ∘ (τ ∘ σ)
    assocS ∅S τ σ = refl
    assocS ⟨ γ , t ⟩ τ σ =
      cong₂ ⟨_,_⟩ (assocS γ τ σ) ([∘]t t τ σ)
```

### Step 2.3: Identity Laws

First, helper lemmas for how weakened terms/substitutions interact with extension.

```agda
abstract
  mutual
    -- Weakened type substituted by extension equals original type substituted
    wkTy-sub : ∀ {Γ Δ A} (B : Ty Δ) (τ : Sub Γ Δ) (t : Tm Γ)
             → wkTy B [ ⟨_,_⟩ {A = A} τ t ]T ≡ B [ τ ]T
    wkTy-sub ⋆ τ t = refl
    wkTy-sub ([ C ] s ⇒ r) τ t =
      cong₂ (λ X Y → [ X ] Y ⇒ wkTm r [ ⟨ τ , t ⟩ ]t)
            (wkTy-sub C τ t) (wkTm-lookup s τ t)
        |> λ p → trans p (cong ([ C [ τ ]T ] s [ τ ]t ⇒_) (wkTm-lookup r τ t))
      where _|>_ : ∀ {a b : Set} → a → (a → b) → b
            x |> f = f x

    -- wkTm applied to a term, then substituted by ⟨ τ , t ⟩, equals original term substituted by τ
    wkTm-lookup : ∀ {Γ Δ A} (s : Tm Δ) (τ : Sub Γ Δ) (t : Tm Γ)
                → wkTm s [ ⟨_,_⟩ {A = A} τ t ]t ≡ s [ τ ]t
    wkTm-lookup (var x) τ t = refl
    wkTm-lookup (coh-raw B u v σ) τ t = cong (coh-raw B u v) (wkSub-∘ σ τ t)

    -- wkSub σ applied to ⟨ τ , t ⟩ reduces to σ ∘ τ
    wkSub-∘ : ∀ {Γ Δ Θ A} (σ : Sub Δ Γ) (τ : Sub Θ Δ) (t : Tm Θ)
             → wkSub σ ∘ ⟨_,_⟩ {A = A} τ t ≡ σ ∘ τ
    wkSub-∘ ∅S τ t = refl
    wkSub-∘ ⟨ σ , s ⟩ τ t =
      cong₂ ⟨_,_⟩ (wkSub-∘ σ τ t) (wkTm-lookup s τ t)
```

Now the main identity laws.

```agda
abstract
  mutual
    [idS]T : ∀ {Γ} (A : Ty Γ) → A [ idS Γ ]T ≡ A
    [idS]T ⋆ = refl
    [idS]T ([ B ] t ⇒ u) =
      cong₂ (λ X Y → [ X ] Y ⇒ u [ idS _ ]t) ([idS]T B) ([idS]t t)
        |> λ p → trans p (cong ([ B ] t ⇒_) ([idS]t u))
      where _|>_ : ∀ {a b : Set} → a → (a → b) → b
            x |> f = f x

    [idS]t : ∀ {Γ} (t : Tm Γ) → t [ idS Γ ]t ≡ t
    [idS]t (var x) = lookup-idS x
    [idS]t (coh-raw B u v σ) = cong (coh-raw B u v) (∘idS σ)

    idS∘ : ∀ {Γ Δ} (σ : Sub Γ Δ) → idS Δ ∘ σ ≡ σ
    idS∘ {Δ = ∅} ∅S = refl
    idS∘ {Δ = Δ , A} ⟨ σ , t ⟩ =
      cong₂ ⟨_,_⟩ (trans (wkSub-∘ (idS Δ) σ t) (idS∘ σ)) refl

    ∘idS : ∀ {Γ Δ} (σ : Sub Γ Δ) → σ ∘ idS Γ ≡ σ
    ∘idS ∅S = refl
    ∘idS ⟨ σ , t ⟩ = cong₂ ⟨_,_⟩ (∘idS σ) ([idS]t t)
```

### Step 2.4: Type Preservation

First, we need lemmas that weakening commutes with substitution.

```agda
abstract
  mutual
    -- Weakening commutes with type substitution
    wkTy-[]T : ∀ {Γ Δ A} (B : Ty Δ) (σ : Sub Γ Δ)
             → B [ wkSub {A = A} σ ]T ≡ wkTy (B [ σ ]T)
    wkTy-[]T ⋆ σ = refl
    wkTy-[]T ([ C ] s ⇒ r) σ =
      cong₂ (λ X Y → [ X ] Y ⇒ r [ wkSub σ ]t)
            (wkTy-[]T C σ) (wkTm-[]t s σ)
        |> λ p → trans p (cong ([ wkTy (C [ σ ]T) ] wkTm (s [ σ ]t) ⇒_) (wkTm-[]t r σ))
      where _|>_ : ∀ {a b : Set} → a → (a → b) → b
            x |> f = f x

    -- Weakening commutes with term substitution
    wkTm-[]t : ∀ {Γ Δ A} (s : Tm Δ) (σ : Sub Γ Δ)
             → s [ wkSub {A = A} σ ]t ≡ wkTm (s [ σ ]t)
    wkTm-[]t (var x) σ = lookup-wkSub x σ
    wkTm-[]t (coh-raw B u v τ) σ = cong (coh-raw B u v) (wkSub-∘' τ σ)

    -- Weakening commutes with composition
    wkSub-∘' : ∀ {Γ Δ Θ A} (τ : Sub Δ Θ) (σ : Sub Γ Δ)
             → τ ∘ wkSub {A = A} σ ≡ wkSub (τ ∘ σ)
    wkSub-∘' ∅S σ = refl
    wkSub-∘' ⟨ τ , t ⟩ σ = cong₂ ⟨_,_⟩ (wkSub-∘' τ σ) (wkTm-[]t t σ)

  -- Specialization of `wkSub-∘'` to the canonical weakening `wk = wkSub (idS _)`.
  ∘wk : ∀ {Γ Δ A} (σ : Sub Δ Γ)
      → σ ∘ wkSub {A = A} (idS Δ) ≡ wkSub σ
  ∘wk {A = A} σ =
    trans
      (wkSub-∘' {A = A} σ (idS _))
      (cong
        (wkSub {A = A})
        (∘idS σ))

  -- Reassociate one outer weakening with the substitution it acts on.
  push-wk-onto-sub :
    ∀ {Γ Δ} {B : Ty Δ} (A : Ty Γ) (τ : Sub Δ Γ)
    → wkTy (A [ τ ]T) ≡ A [ τ ∘ wkSub {A = B} (idS Δ) ]T
  push-wk-onto-sub {B = B} A τ =
    trans
      (sym (wkTy-[]T {A = B} A τ))
      (cong
        (λ X → A [ X ]T)
        (sym (∘wk {A = B} τ)))
```

Now the type preservation lemmas.

```agda
abstract
  -- Type of a weakened term
  tyOf-wk : ∀ {Γ A} (t : Tm Γ)
           → tyOf (wkTm {A = A} t) ≡ wkTy (tyOf t)
  tyOf-wk (var x) = refl
  tyOf-wk (coh-raw B u v σ) = wkTy-[]T ([ B ] u ⇒ v) σ
```

## Variables and Quantification

```agda
-- Boolean variable equality
var-eq : ∀ {Γ} → Var Γ → Var Γ → Bool
var-eq vz     vz     = true
var-eq vz     (vs _) = false
var-eq (vs _) vz     = false
var-eq (vs x) (vs y) = var-eq x y

-- Existential quantification over variables
∃-var : (Γ : Ctx) → ((x : Var Γ) → Bool) → Bool
∃-var ∅       f = false
∃-var (Γ , A) f = (f vz) ∨ (∃-var Γ (λ x → f (vs x)))

-- Universal quantification over variables
∀-var : (Γ : Ctx) → ((x : Var Γ) → Bool) → Bool
∀-var ∅       f = true
∀-var (Γ , A) f = (f vz) ∧ (∀-var Γ (λ x → f (vs x)))
```

### Quantifier Lemmas

```agda
∀-var-map-true :
  {Γ : Ctx} {P Q : Var Γ → Bool}
  → ((x : Var Γ) → P x ≡ true → Q x ≡ true)
  → ∀-var Γ P ≡ true
  → ∀-var Γ Q ≡ true
∀-var-map-true {Γ = ∅} h p = refl
∀-var-map-true {Γ = Γ , A} {P} {Q} h p
  rewrite h vz (∧-trueˡ p)
        | ∀-var-map-true (λ x → h (vs x)) (∧-trueʳ p)
  = refl

∀-var-elim :
  {Γ : Ctx} {P : Var Γ → Bool}
  → ∀-var Γ P ≡ true
  → (x : Var Γ)
  → P x ≡ true
∀-var-elim {Γ = ∅} p ()
∀-var-elim {Γ = Γ , A} p vz     = ∧-trueˡ p
∀-var-elim {Γ = Γ , A} p (vs x) = ∀-var-elim (∧-trueʳ p) x

∀-var-const-true :
  {Γ : Ctx}
  → ∀-var Γ (λ _ → true) ≡ true
∀-var-const-true {Γ = ∅} = refl
∀-var-const-true {Γ = Γ , A}
  rewrite ∀-var-const-true {Γ = Γ}
  = refl

∃-var-cong :
  {Γ : Ctx} {P Q : Var Γ → Bool}
  → ((x : Var Γ) → P x ≡ Q x)
  → ∃-var Γ P ≡ ∃-var Γ Q
∃-var-cong {Γ = ∅} h = refl
∃-var-cong {Γ = Γ , A} {P} {Q} h
  rewrite h vz
        | ∃-var-cong {Γ = Γ} {P = λ x → P (vs x)} {Q = λ x → Q (vs x)}
            (λ x → h (vs x))
  = refl

∃-var-all-false :
  {Γ : Ctx} {P : Var Γ → Bool}
  → ((x : Var Γ) → P x ≡ false)
  → ∃-var Γ P ≡ false
∃-var-all-false {Γ = ∅} h = refl
∃-var-all-false {Γ = Γ , A} {P} h
  rewrite h vz
        | ∃-var-all-false (λ x → h (vs x))
  = refl

∃-var-point-false :
  {Γ : Ctx} {P : Var Γ → Bool}
  → (x : Var Γ)
  → ∃-var Γ P ≡ false
  → P x ≡ false
∃-var-point-false {Γ = ∅} () p
∃-var-point-false {Γ = Γ , A} vz     p = ∨-falseˡ p
∃-var-point-false {Γ = Γ , A} (vs x) p = ∃-var-point-false x (∨-falseʳ p)
```

## Smoke Tests

```agda
private
  Γ₁ : Ctx
  Γ₁ = ∅ , ⋆

  Γ₂ : Ctx
  Γ₂ = Γ₁ , ⋆

  test-var-vz : Tm Γ₁
  test-var-vz = var vz

  test-var-vs : Tm Γ₂
  test-var-vs = var (vs vz)

  test-var-eq₁ : var-eq {Γ₂} vz vz ≡ true
  test-var-eq₁ = refl

  test-var-eq₂ : var-eq {Γ₂} vz (vs vz) ≡ false
  test-var-eq₂ = refl

  -- Test: substitution computes on ⋆
  test-⋆[] : ∀ {Γ Δ} {σ : Sub Δ Γ} → ⋆ [ σ ]T ≡ ⋆
  test-⋆[] = refl

  -- Test: identity substitution on types
  test-idS-T : ∀ {Γ} (A : Ty Γ) → A [ idS Γ ]T ≡ A
  test-idS-T = [idS]T

  -- Test: identity substitution on terms
  test-idS-t : ∀ {Γ} (t : Tm Γ) → t [ idS Γ ]t ≡ t
  test-idS-t = [idS]t

  -- Test: composition distributes over substitution
  test-[∘]t : ∀ {Γ Δ Θ} (t : Tm Θ) (τ : Sub Δ Θ) (σ : Sub Γ Δ)
            → t [ τ ]t [ σ ]t ≡ t [ τ ∘ σ ]t
  test-[∘]t = [∘]t
```
